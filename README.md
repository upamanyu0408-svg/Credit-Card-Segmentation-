# Credit Card Customer Segmentation

## Project Overview
Developed a customer segmentation solution for AllLife Bank using unsupervised machine learning techniques. The objective was to identify distinct customer groups based on spending behavior, credit usage, and customer interactions to support targeted marketing and customer retention strategies.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Key Tasks
- Exploratory Data Analysis (EDA)
- Data Cleaning & Preprocessing
- Feature Scaling
- Customer Segmentation
- K-Means Clustering
- Hierarchical Clustering
- Cluster Evaluation & Interpretation

## Key Insights
- Identified distinct customer segments based on credit card usage and spending patterns.
- Compared clustering techniques to determine the optimal customer grouping strategy.
- Analyzed customer characteristics to understand behavioral differences across segments.

## Business Impact
Provided actionable recommendations for personalized marketing campaigns, customer acquisition strategies, and product upselling opportunities. The segmentation framework can help improve customer engagement and maximize marketing effectiveness.
